# Bilder för Postkoll

Denna mapp innehåller alla bilder och ikoner för Postkoll-appen.

## Filer

### `logo.svg`
Huvudlogotypen som visas i headern. 
- Storlek: 48x48px
- Format: SVG
- Färg: Blå (#0b5cff) med vitt kuvert-ikon

### `favicon.svg`
Favicon som visas i webbläsarens flik.
- Storlek: 32x32px
- Format: SVG
- Färg: Blå (#0b5cff) med vitt kuvert-ikon

## Så ändrar du bilderna

1. **Ersätt logotypen**: Byt ut `logo.svg` mot din egen bild
2. **Ersätt favicon**: Byt ut `favicon.svg` mot din egen ikon
3. **Uppdatera färger**: Öppna SVG-filerna i textredigerare och ändra färgkoderna

### Färgschema
- Primär blå: `#0b5cff`
- Vit: `#ffffff`
- Mörk blå: `#07326a`

## Stödda format
- SVG (rekommenderas för skalbarhet)
- PNG (för favicon: 32x32, 64x64, 192x192)
- ICO (för äldre webbläsare)
